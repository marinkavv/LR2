# Як запустити?

1. Відкрий файл **form.html** у браузері.
2. Заповни поля форми (реєстрація або зворотний зв’язок).
3. Натисни **"Надіслати"**.
4. Перевір результат — повідомлення про помилку або успішне відправлення.
